# 📋 TXT 자동 처리 가이드

## ✅ **현재 유일한 권장 방법**

### 1. 한글 파일을 TXT로 저장
- **인코딩**: UTF-8
- **줄바꿈**: Windows(CR+LF) 
- **서식**: 제거
- **저장 위치**: `data/hwp_texts/` 폴더

### 2. 자동 처리 스크립트 실행
```bash
cd food-safety-search
node scripts/txt-to-database-auto.js
```

## 🎯 **작동 원리**
1. TXT 파일에서 텍스트 읽기
2. 섹션 패턴 자동 인식 (`숫자-숫자 제목`)
3. 섹션별 자동 분리
4. Supabase 데이터베이스에 업로드
5. `file_source: 'txt_auto_extraction_perfect'` 태그 지정

## 📊 **성공 사례**
- ✅ **95개 섹션 100% 성공**
- ✅ **원본 한글 텍스트 완벽 보존**
- ✅ **자동 섹션 분리**
- ✅ **실패율 0%**

## ❌ **사용 금지 방법들**
